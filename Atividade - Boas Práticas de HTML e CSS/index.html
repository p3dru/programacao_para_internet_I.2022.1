Já é uma segunda div, dentro de uma mesma section

Esse parágrafo está aqui apenas para deixar clara a diferença entre deixar um texto livre (sem tag) e outro com tag de parágrafo (p).
O texto livre, obedece a estilização da class div na qual ele está inserido (class="conteudo_secundario", nesse caso).
Já esse paragrafo, obedece à class="conteudo_secundario_paragrafo", e se nada for adicionado nessa estilização em si, ela segue obedecendo à sua classe superior (class="conteudo_secundario")
